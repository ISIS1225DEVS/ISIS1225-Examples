## Acerca del proyecto

Este repositorio es el libro de cocina para el curso de estructuras de datos y algoritmos (#EDA) en Uniandes, contiene conjuntos de datos y ejemplos de código para el uso de los diferentes ADT en la biblioteca DISCLib.

**IMPORTANTE** Este es un trabajo en progreso y es parte de un marco pedagógico para estudiantes universitarios de pregrado en la Universidad de los Andes. Este proyecto NO está pensado como un proyecto de código fuente completamente funcional.

### Estructura

Este repositorio tiene la siguiente estructura:

1. [DISClib](./DISClib) Carpeta raíz con la biblioteca oficial del curso. Para obtener más información sobre su implementación, visite el [Repositorio DISClib][disclib-url].
1. [Examples](./Examples) Carpeta con los scripts de Python. Aquí los estudiantes pueden consultar ejemplos de código para los diferentes ADT en la API DISCLib.
    1. [M1](./Examples/M1) Carpeta con los scripts de Python para el primer módulo del curso que consta de ejemplos para el ADT Lista con Arreglos, Listas Enlazadas, Cola, Pila y los algoritmos de ordenación.
        1. [Leer archivo CSV](./Examples/M1/01-\--csv_fread.py) Script de Python con un ejemplo de cómo leer un archivo CSV y almacenar los datos en una lista.
        1. [List use](./Examples/M1/02-\--list.py) Script de Python con un ejemplo de cómo usar el ADT Lista (arreglo y lista sencillamente enlazada).
        1. [Sortings](./Examples/M1/03-\--sortings.py) Script de Python con un ejemplo de cómo usar los algoritmos de ordenación en la biblioteca DISCLib.
        1. [Queue use](./Examples/M1/04-\--queue.py) Script de Python con un ejemplo de cómo usar la cola de ADT con ADT List.
        1. [Stack use](./Examples/M1/05-\--stack.py) Script de Python con un ejemplo de cómo usar la pila ADT con ADT List.
    1. [:construction: M2 :construction:](./Examples/M2) Carpeta con los scripts de Python para el segundo módulo del curso que consta de ejemplos para el ADT mapa con las técnicas de resolución de colisiones Encadenamiento separado y sondeo lineal, y un ejemplo de uso con tablas hash y listas.
        1. [:construction: Map use :construction:](./Examples/M2/01-\--maps.py) Script de Python con un ejemplo de cómo usar del ADT mapa con sondeo lineal y encadenamiento separado.
        1. [:construction: Hash Tables + Lists :construction:](./Examples/M2/02-\--maps_lists.py) Script de Python con un ejemplo de cómo usar las tablas Hash del mapa ADT con el ADT lista.
    1. [:construction: M3 :construction:](./Examples/M3) Carpeta con los scripts de Python para el tercer módulo del curso que consta de ejemplos para el ADT mapa ordenado (OMap) con las implementaciones de árbol de búsqueda binario (BST) y árbol rojo-negro (RBT), y un ejemplo usando árboles binarios, listas y tablas hash.
        1. [:construction: OMap use :construction:](./Examples/M3/01-\--omaps.py) Script de Python con un ejemplo de cómo usar el ADT OMap con las implementaciones BST y RBT.
        1. [:construction: Tree Search :construction:](./Examples/M3/02-\--tree_search.py) Script de Python con un ejemplo de cómo realizar búsquedas en las implementaciones Binary Search Tree (BST) y Red-Black Tree (RBT).
        1. [:construction: OMaps + ADTs :construction:](./Examples/M3/03-\--omaps_adts.py) Script de Python con un ejemplo de cómo usar las implementaciones BST y RBT en el ADT OMap con la lista ADT y la tabla hash del mapa ADT.
        1. [:construction: MinPQ use :construction:](./Examples/M3/04-\--minpq.py) Script de Python con un ejemplo de cómo usar el ADT MinPQ de la biblioteca DISClib.
    1. [:construction: M4 :construction:](./Examples/M4) Carpeta con los scripts de Python para el cuarto módulo del curso que consta de ejemplos para el gráfico ADT con la lista de adyacencia, varios algoritmos de gráficos como Dijkstra, Bellman-Ford y Kruskal, y un ejemplo que usa gráficos, listas y tablas hash.

1. [:construction: Data :construction:](./Data) Carpeta con archivos relevantes sobre los datos utilizados en los diferentes scripts de Python. 
    1. [Samples](./Data/Samples) Carpeta con archivos de datos CSV para cargar en los diferentes scripts de Python. Aquí los estudiantes pueden leer directamente los datos de cada archivo para comprender su uso en los scripts de ejemplo.
        1. [Pokemon](./Data/Samples/Pokemon-utf8-sample.csv) archivo CSV con una muestra del conjunto de datos de Pokemon.


1. [:construction: Docs :construction:](./Docs) Carpeta con documentos relevantes y los perfiles de datos de los CSV para los diferentes ejemplos de los ADT.
    1. [Perfiles de datos](./Docs/Data-Profiles.pdf) archivo PDF con los perfiles de datos para los diferentes ejemplos de ADT.

## Uso

Para utilizar este repositorio, debe seguir las siguientes recomendaciones:

* Bifurque (Fork) el repositorio a tu propia cuenta de GitHub.
* Clone (Clone) el repositorio en su equipo local.
* Abra el repositorio en su IDE favorito (preferimos VS Code).
* Ejecute los scripts de Python en la carpeta [Examples](./Examples). Cada módulo del curso tiene su propia carpeta con los ejemplos correspondientes. Para obtener información detallada sobre cada script de ejemplo, consulte la sección [Structure](./Structure).
* Para obtener información detallada sobre los conjuntos de datos, consulte las carpetas [Datos](./Datos) y [Documentos](./Docs).
* Para obtener información detallada y documentación de los diferentes ADT, visite el [repositorio DISCLib][disclib-url].
